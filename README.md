# bl4d3.dev

Personal portfolio and website made with [Astro](https://astro.build).

## 🏗️ Devlopment

**Requirements**:

1. [PNPM](https://pnpm.io)
2. [Node.js LTS](https://nodejs.org)

Clone the repository, install the dependencies with `pnpm install` and run `pnpm dev`. It'll spin up a development server on `http://localhost:4321`.

## 🚀 Deployment

`bl4d3.dev` runs statically on Cloudflare Pages, deployed manually with Wrangler.
